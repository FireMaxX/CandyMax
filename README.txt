This is a very simple DIY Candy_Crush Android Game.
All you need to do is choose a fruit and drag it to any of the for directions to swap.

More Details: https://github.com/FireMaxX/CandyMax/blob/master/UserManual.pdf
Download APK file for installation: https://github.com/FireMaxX/CandyMax/blob/master/CandyMax.apk
